Library Management System (C++)

Небольшое консольное приложение для управления личной библиотекой.
Реализовано на языке C++ с применением основных принципов ООП: классы, инкапсуляция, работа с исключениями, указателями и файловым вводом/выводом.
Программа работает через интерактивное меню и позволяет добавлять книги, регистрировать пользователей, выдавать и возвращать литературу.

Основные возможности:

1) Просмотр всех книг

Отображение полного каталога: название, автор, год издания, ISBN, доступность, кто взял книгу.

2) Просмотр всех пользователей

Показ профилей пользователей: фамилия, ID, список взятых книг и лимит.

3) Добавление новой книги

Ввод названия, автора, года и ISBN.

Год автоматически проверяется на корректность (1450 — текущий год).

4) Регистрация пользователя

Добавление пользователя с фамилией, уникальным ID и лимитом на количество книг.

5) Выдача книги

Проверяется:
 • доступна ли книга,
 • существует ли пользователь,
 • не превышен ли лимит.

6) Возврат книги

Книга становится доступной, а у пользователя удаляется из списка взятых.

7) Поиск по ISBN

Быстрый вывод информации по конкретной книге.

9) Просмотр профиля пользователя

Отображение его данных и списка активных займов.

10) Сохранение данных в файл

Записывает текущее состояние библиотеки в текстовый файл.

Технические особенности:
 - Количество книг и пользователей не ограничено
 - Используются стандартные библиотеки:
iostream, fstream, string, vector, stdexcept, sstream
 - Все данные загружаются при старте программы
 - Валидация ввода и обработка исключений
 - Проверка уникальности ISBN
 - Поиск объектов через указатели
 - Сохранение и загрузка состояния из текстового файла
 - Программа написана с использованием трёх основных классов: Book, User, Library
